Dominios

Un dominio de Internet es un nombre único que identifica a un sitio web en Internet. El sistema de nombres de dominios (DNS = Domain Name System) fue creado en 1980 y este traduce las direcciones IP de cada nodo activo en un termino memorizable y facil de encontrar.

Asi los usuarios acceden a un dominio por el nombre (http://example.com) en vez de la dirección IP (http://192.0.32.10). Además una misma dirección IP es compartida por varios nombres de dominios, lo cual incremente la cantidad de posibildades de dominios a nivel mundial.
